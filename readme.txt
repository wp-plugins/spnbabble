SPNBabble posts updates to SPNBabble (a microblog) when you publish a blog entry.

== Description ==

If you have an accounton SPNBabble (http://spnbabble.sitepronews.com) you can post
and update to your account each time you publish a new blog entry.

There are options to enable/disable the updates to give you more control over which blog
entries are posted to SPNBabble.

== Installation ==

Upload the SPNBabble plugin to your blog, Activate it, then enter your SPNBabble Username
and SPNBabble Password and fill in your Blog Name.
